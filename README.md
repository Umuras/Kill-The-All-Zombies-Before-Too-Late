# Kill The All Zombies Before Too Late
### Oyunun Amacı <br/>
Bu oyunda amacımız 60 saniye içerisinde bir dalgada oluşan tüm zombileri zaman bitmeden öldürmek.
### Oyundaki Özellikler
- 2 tane oynanabilir bölüm, Training Level ve Main Stage
- Training Level
